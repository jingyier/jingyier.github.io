HTML基础知识

  • HTML 代码是由 “标签” 构成的；
  • 整体的 HTML 代码是由一对 html 包裹的；
  • head 标签是页面的头部，标签中写页面的属性；
  • body 标签是页面的主体，标签中写的是页面显示的内容；
  • title 标签中写的是页面的标题。
